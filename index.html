Hướng dẫn sử dụng

Mỗi dự án animation có 2 liên kết chính:

  • Xem Demo: Mở trang GitHub Pages để xem animation trực tiếp
  • Xem Code: Mở trang GitHub để xem mã nguồn của dự án

Đảm bảo rằng mỗi thư mục đã được cấu hình GitHub Pages riêng để "Xem Demo" hoạt động chính xác.
